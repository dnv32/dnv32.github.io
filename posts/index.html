    Multiple Input Multiple Output Switch aka MIMOSA went open source!

    September 22, 2020
    electronics, mechanics, wearables
    professional

    MIMOSA is designed to be the most versatile user interface for wide veriety of applications. The project just is now open-source hardware and the materials are available on Github.

    The hardware is based on versatile nRF52840, with rich libraries support. Also can be configured to work with CircuitPython such that it is very easy to configure for everyone! More firmware and software work will be available soon.
